Access-It Remote Desktop

Secure screen sharing and remote control

About

Access-It is a lightweight remote desktop application that allows screen sharing and remote control between two users. It's designed to be simple, secure, and efficient.

Features

  • Screen Sharing: Host shares their screen with the client in real-time
  • Remote Control: Client can control the host's computer via mouse and keyboard
  • Simple Authentication: Uses a 6-digit session code for connection
  • Secure Communication: All data is encrypted using WebRTC
  • Cross-Network Compatible: Works across different networks using STUN/TURN servers

Getting Started

To use Access-It, you need to download and install the client application on both computers:

  1. Download the Access-It client for your platform
  2. Install and run the application
  3. Choose whether you want to be the host (share your screen) or the client (view and control)
  4. Follow the on-screen instructions to connect

How It Works

Access-It uses WebRTC for peer-to-peer communication, which means:

  • Your screen data goes directly from host to client, not through our servers
  • All communication is encrypted end-to-end
  • Low latency for responsive remote control
  • Works across different networks and firewalls
